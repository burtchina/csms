## 系统简介

校园安全管理系统(CSMS)是一款专为学校安全管理部门设计的综合性管理平台。系统提供设备管理、故障监控、维护排期、性能分析等功能，帮助管理人员高效管理校园安全设备，提升校园安全水平。

## 登录与注册

### 登录系统

1. 在浏览器中访问系统地址：`http://localhost:5000`（或您的系统部署地址）
2. 在登录页面输入用户名和密码
3. 勾选"记住我"可在一周内保持登录状态
4. 点击"登录"按钮进入系统

![登录界面](images/login.png)

### 注册账号

1. 在登录页面点击"没有账号？去注册"链接
2. 填写用户名、电子邮箱和密码
3. 确认密码后点击"注册"按钮
4. 管理员审核通过后即可使用新账号登录系统

### 忘记密码

1. 在登录页面点击"忘记密码"链接
2. 输入注册时使用的电子邮箱
3. 点击"发送重置链接"
4. 系统将发送密码重置邮件，按照邮件中的说明操作

## 系统主界面

成功登录后，您将看到系统主界面，包括以下部分：

1. **顶部导航栏**：包含系统名称、主要功能模块入口和用户菜单
2. **仪表盘**：展示系统概况和重要指标
3. **快速导航卡片**：快速访问常用功能
4. **通知区域**：显示系统通知和提醒

## 设备管理

设备管理模块用于管理校园内所有安全设备的信息、状态和配置。

### 设备列表

设备列表页面显示系统中所有设备的基本信息，包括：

- 设备名称
- 设备类型
- 设备状态
- 位置信息
- 最后维护时间

您可以通过搜索栏、筛选器和排序功能快速找到需要的设备。

### 添加设备

1. 在设备列表页面点击"添加设备"按钮
2. 填写设备基本信息（名称、类型、型号等）
3. 填写设备网络信息（IP地址、MAC地址等）
4. 填写设备位置信息
5. 填写设备维护信息
6. 点击"保存"按钮完成添加

### 设备详情

点击设备列表中的设备名称，进入设备详情页面，您可以：

- 查看设备详细信息
- 查看设备历史故障记录
- 查看设备维护历史和计划
- 查看设备性能数据图表
- 编辑设备信息
- 删除设备

## 故障管理

故障管理模块用于处理设备故障报告、跟踪故障处理进度和分析故障数据。

### 故障列表

故障列表页面显示所有设备故障记录，包括：

- 故障ID
- 设备名称
- 故障类型
- 故障描述
- 报告时间
- 故障状态
- 处理人员

### 报告故障

1. 在故障列表页面点击"报告故障"按钮
2. 选择发生故障的设备
3. 选择故障类型
4. 填写故障描述
5. 设置优先级
6. 上传相关图片（可选）
7. 点击"提交"按钮

### 处理故障

1. 在故障列表中找到需要处理的故障记录
2. 点击"处理"按钮
3. 更新故障状态
4. 添加处理说明
5. 上传处理结果图片（可选）
6. 点击"保存"按钮

## 维护管理

维护管理模块用于规划和记录设备的定期维护工作。

### 维护计划

维护计划页面显示所有设备的维护计划，包括：

- 维护ID
- 设备名称
- 维护类型
- 计划时间
- 预计持续时间
- 负责人
- 状态

### 创建维护计划

1. 在维护计划页面点击"创建计划"按钮
2. 选择需要维护的设备
3. 选择维护类型
4. 设置计划时间和持续时间
5. 指定负责人
6. 添加维护说明
7. 点击"保存"按钮

### 记录维护结果

1. 在维护计划列表中找到已完成的维护任务
2. 点击"记录结果"按钮
3. 填写维护过程和结果
4. 更新设备状态
5. 上传相关图片（可选）
6. 点击"提交"按钮

## 性能监控

性能监控模块用于实时监控和分析设备性能数据。

### 监控仪表盘

监控仪表盘展示设备关键性能指标，包括：

- 设备在线率
- 网络连接质量
- 存储空间使用情况
- CPU/内存使用率
- 温度等环境参数

### 设置告警规则

1. 在性能监控页面点击"告警规则"标签
2. 点击"添加规则"按钮
3. 选择设备和监控指标
4. 设置阈值和触发条件
5. 设置通知方式（邮件、短信等）
6. 点击"保存"按钮

## 常见问题

### 无法登录系统

**问题**：输入正确的用户名和密码后无法登录

**解决方案**：
1. 确认用户名和密码是否正确
2. 确认账号是否已被管理员启用
3. 清除浏览器缓存后重试
4. 联系管理员重置密码

### 无法添加设备

**问题**：添加设备时提示"IP地址已存在"

**解决方案**：
1. 检查输入的IP地址是否正确
2. 在设备列表中搜索该IP地址，确认是否已有设备使用
3. 如确认无误，可尝试使用不同的IP地址或联系管理员处理

### 系统性能问题

**问题**：系统响应速度变慢

**解决方案**：
1. 清除浏览器缓存
2. 检查网络连接状况
3. 如问题持续，请联系系统管理员检查服务器状态
